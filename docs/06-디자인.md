## 무드 선언 (이전 논의)

> "디지털로 만드는 아날로그 앨범. 만든 티 안 나는, 그래서 더 예쁜."

인스타의 과잉 큐레이션도 아니고, BeReal의 날 것 그대로도 아닌 — 핀터레스트 감성의 스크랩북. 카카오/네이버 느낌은 철저히 배제.

## 비주얼 언어 (앨범 페이지)

물리적 스크랩북을 디지털로 재현한 느낌:

- 사진이 살짝 기울어질 수 있음 (±3~5°), 자연스럽게 겹침
- 배경: 미세한 그레인의 종이 텍스처
- 스티커: 실제로 붙인 느낌의 마이크로 그림자
- 사진 테두리: 폴라로이드, 빈티지 프레임 선택 가능
- 네비게이션 바/모달: Liquid Glass 적용

---

## 마이크로인터랙션 (차별화 포인트)

- 사진 추가: 블러 → 선명해지는 현상 과정 애니메이션 (0.8초)
- 스티커 배치: 착지 시 물리적 바운스
- 페이지 넘기기: 실제 종이 컬 이펙트
- 공유앨범 초대 수락: 봉투 열리는 애니메이션

## 피해야 할 것

- 순수 플랫 디자인 (시대에 뒤떨어짐)
- 차가운 미니멀리즘 (카카오/네이버 느낌)
- AI 생성 일러스트 (트렌드가 실제 사진으로 이동 중)
- 과잉 맥시멀리즘

## 적극 활용해야 할 것

- 레이어드 깊이감 (섬세한 그림자, 겹치는 요소)
- 마이크로인터랙션 (사진 배치, 데코 적용 순간)
- 둥근 코너 (친근하고 현대적)
- 다크 모드 (기본 스펙)
- Bento Grid (앨범 브라우징)
- Kinetic typography (핵심 순간에 적용)
