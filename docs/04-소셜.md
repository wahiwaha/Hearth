# 04. 소셜

> 최종 수정: 2026-03-02

---

## 개요

타인의 작업물 중 타인이 공유한/전체공개로 표시한 작업물을 볼 수 있음.

---

## 탐색 피드

- 돋보기 아이콘을 누르면 다른 사람이 만든 **앨범 표지 / 한 페이지(폴리오) / 개별 사진**이 보임
- 셋 중에 뭘 볼지는 **필터**를 통해 고를 수 있음

---

## 콘텐츠 상세 보기

- 보고싶은 객체를 터치하면 확대되어 화면에 나타남
- 인스타와 유사하나, 인스타처럼 전체화면으로 커지지 않음
- **맥처럼 애니메이션과 함께 전체화면의 약 80% 크기로 중앙에 나타남**
- 그 이후 만든 사람, 만든 날짜, 사진이 찍힌 위치 등이 멋지게 보임
  - 물론 모두 사용자가 사진에 메모/태그한 내용에 한함

---

## 템플릿 시스템

- 만든 사람이 템플릿 공유 허용했다면 템플릿을 확인할 수 있음
- 템플릿(사진을 제외한 디자인)은 저장하여 나중에 앨범을 꾸밀 때 불러올 수 있음

### 편집 가능 여부

- **앱 내에서 만들어진 디자인:** 편집 가능
  - 이미지 움직이기, 배경색 바꾸기, 이미지 삭제 등
- **앱 외부에서 만들어진 디자인:** 편집 불가
  - 외부에서 꾸며진 배경을 가져와서 배경이미지 삽입 방식으로 한번에 적용했다면, 거기서 특정 요소만을 추출하거나, 색상을 바꾸거나 할 수 없음

---

## 공개범위 설정

### 망원경 아이콘

사용자는 앨범/앨범 페이지/사진 각각을 볼 때 나타나는 **우측 위 망원경 아이콘**을 눌러 공개범위를 설정할 수 있다.

### 공개범위 옵션

- **나만 공개** (기본)
- **친구에게 공개**
- **전체 공개**

### 컨텍스트별 동작

| 현재 보고 있는 화면 | 망원경 아이콘 누르면 |
|---------------------|---------------------|
| 앨범 라이브러리 (앨범 이름과 함께 표지들이 보이는 페이지) | 공유할 앨범을 선택하여 공개범위 설정 |
| 앨범 내부 (페이지 목록 보는 중) | 앨범 페이지의 공개범위 설정 |
| 개별 페이지 보는 중 | 해당 페이지의 공개범위 설정 |
| 사진 보는 중 | 해당 사진의 공개범위 설정 |
| 갤러리처럼 여러 페이지를 한번에 보는 중 | 사진을 **여러 장씩 선택**한 후 공개범위 지정 가능 |

> *기본적으로 공개범위와 관련된 UI/UX/로직은 다른 소셜미디어를 참고하라

## 다른 방안 1: 인스타 스타일 탐색

### 탐색 피드

- 인스타처럼 탐색 피드에는 **필터를 적용하지 않으면 개별 사진만 보임**
- 개별 사진을 누른 후 들어가면:
  - 만든 사람
  - 만든 날짜
  - 템플릿 저장 (가능하다면)
  - 사용자가 폴리오(페이지)나 앨범 커버 공유를 허용했다면 **'앨범 보기' 버튼**이 보임

### 앨범 보기

- 앨범 보기 버튼을 누르면, 클릭했던 사진이 있는 폴리오가 보임
- **양옆으로 드래그**하면서 앨범을 둘러볼 수 있음
- 확대/축소 하면서 여러 폴리오씩 보기 가능

### 공개 범위와의 연동

- 보이는 것의 범위는 사용자가 공개허용한 범위와 같음
- **예시:** 100장의 폴리오 중 5개+커버만 공개했다면, 타자가 볼 수 있는 건 총 7페이지 (커버 2장 + 폴리오 5장)

### 블러 처리

- 한 폴리오에 있는 여러 사진들 중 개인이 가린 게 있다면, 폴리오는 보여도 폴리오 속 사진은 **강한 블러처리**가 된 상태로 보임
