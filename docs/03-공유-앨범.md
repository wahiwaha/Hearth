# 03. 타인과의 앨범 (공유 앨범)

> 최종 수정: 2026-03-02

---

## 개요

공동작업자와 같이 꾸미는 앨범.

---

## 상세

- 앨범을 만들 때 **공동작업자를 추가**하게 되면, **실시간 편집 기능**이 모두에게 주어지고, 앨범도 **모두의 라이브러리에 생김**
- 거기서부터 둘이 같이 찍힌 사진을 앨범에 추가하며 앨범을 꾸밀 수 있음
- 앨범을 같이 꾸미고자 하는 타인이 **앱 사용자가 아니라면**, 타인에게 각종 방법으로 메시지를 보낼 수 있음
  - 애플에서 제공하는 내보내기 버튼
  - 초대번호
  - 등등의 방식

---

## 사진 업로드 플로우 (AI 연동)

1. 사진 촬영 또는 업로드
2. AI가 얼굴 감지 → 이미 등록된 친구면 "○○과의 앨범에 추가할까요?" 제안
3. 처음 보는 얼굴이면 "이 사람 누구예요?" 질문 → 태깅
4. 해당 인물이 앱 유저면 → 공유앨범에 즉시 반영
5. 앱 유저가 아니면 → 카카오톡/문자 초대 메시지 자동 생성 ("같이 추억 만들어가자~" 방식)
6. 이 시점부터 사진 + 앨범 배경은 둘(또는 그 이상)이 함께 편집하는 공간이 됨

---

## 공동 편집

- 공유 앨범은 A와 친구 모두의 '나의 앨범'에 포함될 수 있다
- 나의 앨범에 포함될 때 수정 가능한 앨범이 되는 조건부 구조

---

## MVP 기능 범위 (Must Have)

- 친구 초대 → 공유앨범 생성 → 공동 편집
- 공유앨범을 나의 앨범에 삽입하는 기능
- 미가입자에게 초대 메시지 발송
