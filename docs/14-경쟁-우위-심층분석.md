## 1. 글로벌 사진 앱 지형도 (2024-2025)

### 주요 플레이어 MAU

| 앱 | MAU | 카테고리 |
|----|-----|---------|
| Instagram | ~30억 (Meta 발표, 2025.09) | 소셜 사진+영상 |
| Snapchat | 9.46억 MAU, 4.77억 DAU | 순간 사진 메시징 |
| Pinterest | 5.37억 | 비주얼 큐레이션 |
| Google Photos | ~20억+ | 클라우드 저장 + AI |
| BeReal | ~1,600만 (하락세) | 인증샷 |
| Locket | 900만 DAU, 3,000만+ 가입 | 위젯 사진 공유 |

### 시장 양극화 현상

시장이 두 갈래로 나뉘고 있다:

1. **메가 플랫폼** (Instagram, Snapchat) — 방송/엔터테인먼트 지향
2. **친밀한 소규모 앱** (Locket, Retro, BeReal) — Gen Z의 "진짜 관계" 니즈 충족

> **빈 자리: "소규모 친밀한 관계 + 앨범/꾸미기"를 소유한 앱은 글로벌/한국 모두 없다.**

## 2. 한국 시장 경쟁자 심층 분석

### 찍스 (Zzixx)

| 항목 | 내용 |
|------|------|
| 포지셔닝 | "대한민국 1등 사진인화" |
| 제품 | 사진 인화, 포토북, 액자, 달력, 스티커 주문 |
| 규모 | 일 25만 장 인화, 누적 6억 장+ |
| 투자 | KRW ~4억 (초기 투자, 비상장) |
| **평가** | **소셜 앱이 아님.** 순수 인화 주문 유틸리티. 꾸미기/소셜 기능 없음. Hearth의 경쟁자가 아니라 **잠재적 파트너.** |

### 패밀리앨범 (FamilyAlbum / Mitene)

| 항목 | 내용 |
|------|------|
| 개발사 | MIXI (일본), 부트스트래핑 |
| 포지셔닝 | 가족 전용 사진 공유 |
| 규모 | 800만+ 유저, 월 ~$300K 매출 |
| **평가** | **육아/가족 특화 니치.** 친구 그룹, 커플, 범용 앨범에는 부적합. 타깃이 완전히 다름. |

### MoStory (모스토리)

| 항목 | 내용 |
|------|------|
| 포지셔닝 | 사진 꾸미기/다이어리 |
| 기능 | 사진 데코, PDF 내보내기, 포토레터 |
| **평가** | **1인용 크리에이티브 툴.** 소셜 그래프 없음, 협업 기능 없음. 디지털 스크랩북 제작 도구에 가까움. |

### Between (비트윈)

| 항목 | 내용 |
|------|------|
| 개발사 | VCNC → Krafton (배그 개발사) 인수 |
| 포지셔닝 | 커플 전용 앱 |
| 규모 | 글로벌 2,600만 다운로드, 100만 DAU (2018 기준) |
| **평가** | **한국에서 가장 유사한 정신적 경쟁자이나 커플(2인)에 한정.** 친구 그룹, 가족 앨범에는 부적합. |

### Retro (미국, 한국 진출 중)

| 항목 | 내용 |
|------|------|
| 창업자 | ex-Instagram 팀 (Stories 출시 리드) |
| 투자 | ~$18M (Sequoia, Thrive Capital) |
| 포지셔닝 | 친한 친구와의 주간 사진 저널 |
| **평가** | **글로벌에서 가장 직접적인 경쟁자.** 단, 꾸미기/인화 기능 없음. 한국 시장 로컬라이즈 미흡. |

### 기타

- **사이사이:** 커플 앨범 앱 (2024 출시), 프라이버시 중심
- **럽다:** 커플 데이팅 앱 + 앨범
- **Mellog:** 커플 다이어리, 필름 감성
- **SNAPS/오프린트미:** 인화 서비스 → 경쟁자가 아닌 **파트너**

## 3. 왜 대형 플레이어가 이 영역을 못 먹는가 (Innovator's Dilemma)

### Instagram: 구조적으로 앨범 시장 진입 불가

| 요인 | 설명 |
|------|------|
| **매출 구조** | Meta 연 $65B+ 광고 매출. 5~10명의 비공개 앨범은 광고 노출 0회 → 매출 기여 제로 |
| **전략 방향** | 2023년 Shop 탭 제거, Reels에 올인 (스크린타임 35%). 앨범은 전략 반대 방향 |
| **문화적 부적합** | Instagram은 "보여주기" 플랫폼. 앨범은 "간직하기" → 정반대 가치관 |
| **자기잠식 우려** | 비공개 앨범 기능 추가 시 공개 피드 체류시간 감소 → 광고 매출 하락 |

### Apple Photos / Google Photos: 유틸리티 DNA

| 요인 | Apple Photos | Google Photos |
|------|-------------|---------------|
| **핵심 방향** | 온디바이스 AI, 프라이버시, 생태계 | 저장/백업, Gemini AI |
| **소셜 기능** | 공유 라이브러리 5인 제한, 3MP 열화 | 협업 추억 (2023~), 패시브 |
| **소셜 DNA** | 없음. 모더레이션/커뮤니티 관리 역량 부재 | Google+, Buzz, Allo, Duo 전부 실패 |
| **결론** | 소셜 네트워크 빌드 의지도 역량도 없음 | 유틸리티 인식 전환 극도로 어려움 |

> Google Photos는 2024년 7월에 Retro의 디자인을 그대로 베낀 것으로 공개 비판받음 (TechCrunch). 이는 자체 소셜 기능 혁신 역량 부재를 방증.

### SNOW / Zepeto: 완전히 다른 방향

| 요인 | SNOW | Zepeto |
|------|------|--------|
| **핵심** | 셀피 카메라, 뷰티 필터, AR | 메타버스, 아바타, 브랜드 파트너십 |
| **직원** | ~367명 | SNOW Corp 산하 |
| **전략** | 개인을 더 예쁘게 → Hearth는 추억을 있는 그대로 보존 | Roblox/Fortnite과 경쟁 |
| **소셜 그래프** | 없음 (카메라 도구) | 있으나 가상 세계 용 |
| **결론** | 철학적으로 정반대. 피봇 동기 없음 | 완전히 다른 카테고리 |

### Clayton Christensen의 Innovator's Dilemma 프레임워크

1. **자원 배분:** Instagram 최고 엔지니어는 Reels, Google은 Gemini, SNOW는 AR에 투입. "작은" 앨범 기능에 핵심 인력 배치는 비합리적.
2. **고객 청취 함정:** 기존 유저들은 더 나은 필터, 더 빠른 영상을 원함. 앨범을 원하는 유저는 다른 세그먼트.
3. **마진 구조:** 앨범 앱의 구독 $2-5/월은 Instagram의 $65B 광고 매출 관점에서 의미 없는 규모. 하지만 독립 스타트업에게는 거대한 사업.
4. **조직 관성:** 대기업의 KPI, 승진 기준, 리뷰 사이클 모두 이 방향을 지원하지 않음.

## 4. Hearth의 경쟁 해자 (Moat) 구조

### 3중 해자 모델

```
┌─────────────────────────────────┐
│    1. 콘텐츠 락인 (Content)      │  ← 사진, 앨범, 꾸미기 투자
│  ┌───────────────────────────┐  │
│  │  2. 소셜 그래프 (Social)   │  │  ← 공유 앨범 속 관계
│  │  ┌─────────────────────┐  │  │
│  │  │ 3. 감정 락인 (Emotion)│  │  │  ← 대체 불가한 추억
│  │  └─────────────────────┘  │  │
│  └───────────────────────────┘  │
└─────────────────────────────────┘
```

**1. 콘텐츠 락인:**
수백 장의 사진, 꾸며진 앨범, 캡션, 데코레이션 → 다른 플랫폼으로 이전 비용 극대

**2. 소셜 그래프 락인:**
8명의 가족, 5명의 친구가 공유 앨범에 참여 → 전원이 동시에 이동해야 함 → 사실상 불가능 (카카오톡과 동일 메커니즘)

**3. 감정 락인:**
고인의 사진, 결혼식, 졸업, 유년기 추억 → 대체 불가한 감정적 가치. 사용자는 이 추억을 잃을 위험을 감수하지 않음.

### 네트워크 효과 분석

| 유형 | Hearth에서의 작동 방식 |
|------|----------------------|
| **직접 네트워크 효과** | 앨범에 사람이 많을수록 → 더 많은 사진 → 더 풍성한 앨범 → 더 높은 가치 |
| **간접 네트워크 효과** | 유저가 많을수록 → 더 다양한 템플릿/디자인 공유 → 꾸미기 생태계 성장 |
| **데이터 네트워크 효과** | 사진이 많을수록 → AI 추천 정확도 향상 → 더 나은 자동 앨범 제안 |

### 바이럴 계수 (K-Factor) 분석

- 공유 앨범 1개 생성 시 평균 3~10명 초대
- 30~50% 전환 가정 시 **K-factor = 0.9~5.0**
- 핵심: **초대 → 첫 사진 업로드까지의 시간(바이럴 사이클 타임)을 최소화**하는 것이 관건

### 한국 소셜앱 바이럴 성공 사례에서 배우는 교훈

| 앱 | 핵심 바이럴 메커니즘 | Hearth 적용 |
|----|---------------------|------------|
| **카카오톡** (2010) | 전화번호부 기반 즉시 소셜 그래프 + 무료 (SMS 대비) | 연락처 기반 친구 찾기 + 앨범 초대 |
| **당근마켓** | 하이퍼로컬 신뢰 + 거래의 즉각적 가치 | "가까운 관계"에 집중 → 양보다 질 |
| **Locket** | TikTok 내부 팀 매일 콘텐츠 → 오가닉 바이럴 | 앨범 꾸미기 과정 자체가 숏폼 콘텐츠가 됨 |

## 5. 경쟁 포지셔닝 맵

```
                    소셜 (공유/협업)
                         ↑
                         │
          Retro ●        │        ● Instagram
                         │
     Hearth ★            │
                         │
  FamilyAlbum ●          │        ● Snapchat
                         │
  ─────────────────────────────────────→ 꾸미기/크리에이티브
  앨범/보존 지향          │        엔터/소비 지향
                         │
     MoStory ●           │        ● SNOW/B612
                         │
     Between ●           │        ● BeReal
                         │
                   개인 (비공유)
```

> **Hearth가 차지하는 좌상단 포지션 (소셜 + 앨범/보존 + 꾸미기)은 현재 비어있다.**

## 6. 결론: 사업 정당성

1. **시장 공백이 실재한다:** "공유 앨범 + 꾸미기 + 소셜"을 통합한 서비스는 국내외 모두 없음
2. **대형 플레이어가 진입하기 어렵다:** Innovator's Dilemma가 구조적으로 작용
3. **해자가 시간과 함께 강화된다:** 콘텐츠·소셜·감정의 3중 락인
4. **비교 기업들이 VC 투자를 받고 있다:** Lapse($150M 밸류), Retro(Sequoia 투자), BeReal($538M 인수)
5. **한국발 글로벌 확장의 검증된 경로가 있다:** Zepeto(95% 해외), SNOW(2억+ 글로벌)
