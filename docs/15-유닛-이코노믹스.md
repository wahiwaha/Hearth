## 1. 수익 모델 구조

Hearth는 **4개의 수익원**을 가진 하이브리드 모델:

```
┌──────────────────────────────────────────────┐
│              Hearth 수익 구조                  │
├──────────────┬───────────────────────────────┤
│  반복 수익    │  ① 프리미엄 구독 (SaaS)       │
│  (Recurring)  │  ② 템플릿/디자인 마켓플레이스  │
├──────────────┼───────────────────────────────┤
│  거래 수익    │  ③ 포토북/인화 주문 (커머스)    │
│  (Transact.)  │  ④ 브랜드 콜라보/시즌 한정     │
└──────────────┴───────────────────────────────┘
```

## 2. 수익원별 상세

### ① 프리미엄 구독 (월 ₩4,900 / 연 ₩47,000)

| 무료 (기본) | 프리미엄 |
|-------------|---------|
| 앨범 3개 생성 | 앨범 무제한 |
| 기본 템플릿 | 프리미엄 템플릿 전체 |
| 기본 스티커/배경 | 프리미엄 스티커/배경/프레임 전체 |
| 공유 앨범 1개 | 공유 앨범 무제한 |
| 사진 저장 1GB | 사진 저장 50GB |
| 기본 AI 추천 | 고급 AI 자동 앨범 + 얼굴인식 |
| 워터마크 포함 내보내기 | 워터마크 없는 고해상도 내보내기 |

**벤치마크:** Locket Pro ~$4.99/월, Between Premium ~₩4,900/월

### ② 템플릿/디자인 마켓플레이스

- 크리에이터가 만든 앨범 템플릿을 판매 (₩1,000~₩5,000)
- Hearth 수수료 30% (앱스토어 모델)
- 시즌별 공식 템플릿 팩 (벚꽃, 크리스마스, 졸업 등)
- 소셜에서 발견한 템플릿 → 즉시 구매 → 내 앨범에 적용

### ③ 포토북/인화 주문 (핵심 BM)

| 상품 | 예상 가격 | 마진율 |
|------|----------|--------|
| 소프트커버 포토북 (20p) | ₩15,000~25,000 | 40~50% |
| 하드커버 포토북 (40p) | ₩35,000~55,000 | 45~55% |
| 개별 사진 인화 (10장) | ₩5,000~8,000 | 50~60% |
| 포토카드 세트 (24장) | ₩12,000~18,000 | 45~55% |
| 앨범 커버 액자 | ₩20,000~35,000 | 40~50% |

**제조 파트너:** SNAPS, 찍스 등 기존 인화 인프라 활용 (직접 인프라 구축 불필요)

> 핵심 인사이트: 앱에서 이미 꾸며진 앨범이 존재 → "이걸 실물로 뽑고 싶다"는 자연스러운 전환. 기존 인화 서비스 대비 **디자인이 이미 완성된 상태**에서 주문이 발생하므로 전환율이 월등히 높을 것.

### ④ 브랜드 콜라보

- 시즌 한정 템플릿/스티커 팩 (예: 스타벅스 x Hearth 크리스마스 앨범)
- 브랜드 포토카드 (K-pop 아이돌 콜라보 등)
- 위치 기반 스폰서드 포토스팟

## 3. 유닛 이코노믹스

### 핵심 가정

| 지표 | 보수적 | 낙관적 | 근거 |
|------|--------|--------|------|
| CAC (유저 획득 비용) | ₩5,000 | ₩3,000 | 바이럴 중심, 초기 인스타 마케팅 병행 |
| 월 구독 전환율 | 5% | 8% | 업계 평균 2-5%, 꾸미기 앱은 높은 편 |
| 구독 이탈률 (월) | 8% | 5% | 감정 락인으로 업계 평균(10-15%) 대비 낮을 것 |
| 구독자 평균 수명 | 12.5개월 | 20개월 | 1/이탈률 |
| 인화 주문 전환율 (연) | 3% | 5% | MAU 대비 연간 1회 이상 주문 비율 |
| 평균 주문가 (AOV) | ₩25,000 | ₩35,000 | 포토북 기준 |
| 인화 마진율 | 40% | 50% | 제조 파트너 위탁 기준 |

### LTV (고객 생애 가치) 계산

**보수적 시나리오:**

```
구독 LTV = ₩4,900 × 12.5개월 = ₩61,250
인화 LTV = ₩25,000 × 0.4 마진 × 1.5회 (수명 내) = ₩15,000
템플릿 LTV = ₩2,000 × 2회 = ₩4,000
──────────────────────────────
총 LTV = ₩80,250 (구독 전환 유저 기준)
```

**비구독 유저 LTV:**
```
인화 LTV = ₩25,000 × 0.4 × 0.5회 = ₩5,000
템플릿 LTV = ₩2,000 × 1회 = ₩2,000
──────────────────────────────
총 LTV = ₩7,000
```

**블렌디드 LTV (전체 유저 평균):**
```
= (5% × ₩80,250) + (95% × ₩7,000)
= ₩4,013 + ₩6,650
= ₩10,663
```

### LTV/CAC 비율

| 시나리오 | LTV | CAC | LTV/CAC | 판정 |
|----------|-----|-----|---------|------|
| 보수적 | ₩10,663 | ₩5,000 | **2.1x** | 건전 (3x 이상이 이상적) |
| 낙관적 | ₩18,400 | ₩3,000 | **6.1x** | 우수 |

> VC 기준: LTV/CAC > 3x가 투자 기준선. 보수적 시나리오에서도 바이럴 효과가 발생하면 실질 CAC가 ₩2,000 이하로 떨어져 **3x 달성 가능.**

### CAC 페이백 기간

| 시나리오 | 월 ARPU (구독자) | 페이백 기간 |
|----------|-----------------|-------------|
| 보수적 | ₩4,900 | ~1.0개월 (구독자), ~7개월 (블렌디드) |
| 낙관적 | ₩4,900 | ~0.6개월 (구독자), ~4개월 (블렌디드) |
