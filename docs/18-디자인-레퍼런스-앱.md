# 18. 디자인 & 기능 레퍼런스 앱 목록

> 최종 수정: 2026-03-03

---

Hearth의 핵심 기능(공유 앨범, 스크랩북 꾸미기, 소셜 피드, 인화 주문)에 대한 디자인/UX 레퍼런스가 될 수 있는 앱 30개를 카테고리별로 정리한다.

---

## 1. 사진 공유 / 앨범 앱 (직접 경쟁군)

### 1-1. Retro — Photos with Friends
- **카테고리:** 친구 간 사진 공유 저널
- **플랫폼:** iOS, Android
- **참고할 디자인/기능:**
  - **주간 사진 스토리(Weekly Roll):** 일주일 단위로 사진을 자동 큐레이션해 "이번 주의 이야기"로 만들어주는 UX. Hearth의 앨범 자동 생성 흐름에 참고.
  - **Collaborative Journals:** 여러 친구가 하나의 저널에 사진을 함께 올리는 구조 -- Hearth의 공유 앨범 협업 UX의 직접 레퍼런스.
  - **Rewind (되감기):** 작년 이맘때 사진을 다이얼 인터랙션으로 되돌아보는 기능. Hearth의 "추억 재방문" 기능의 인터랙션 참고.
  - **Photo Postcards:** 앱 내에서 친구에게 포스트카드를 보내는 기능 -- 인화와 소셜을 연결하는 UX 참고.
- **수상/인지도:** Apple 2025 Cultural Impact Award 파이널리스트. 12개국 사진 앱 1위. 약 100만 사용자.

### 1-2. Lapse
- **카테고리:** 일회용 카메라 시뮬레이션 사진 공유
- **플랫폼:** iOS, Android
- **참고할 디자인/기능:**
  - **현상 지연(Developing Delay):** 촬영 후 사진이 랜덤 시간에 현상되는 UX. 기다림의 설렘을 만드는 게이미피케이션 참고.
  - **월간 포토덤프(Monthly Photodump):** 자동으로 한 달의 사진을 앨범화하는 흐름. Hearth의 자동 앨범 생성 로직 참고.
  - **36장 제한:** 의도적으로 촬영 수를 제한해 매 사진에 의미를 부여하는 디자인. "소중한 순간"이라는 Hearth 컨셉과 맞닿는 제약 설계.
- **수상/인지도:** 2023년 출시 직후 App Store 1위 달성. 초대 전용 모델로 입소문 성장.

### 1-3. Locket Widget
- **카테고리:** 홈화면 위젯 기반 실시간 사진 공유
- **플랫폼:** iOS, Android
- **참고할 디자인/기능:**
  - **홈화면 위젯 UX:** 앱을 열지 않아도 홈화면에서 친구/연인의 사진이 업데이트되는 패턴. Hearth의 위젯 기능 설계 시 참고.
  - **친밀한 공유 범위(최대 20명):** 소규모 그룹에 집중하는 설계. Hearth의 공유 앨범 인원 설정 UX 참고.
  - **즉각적 반응:** 사진을 보내면 상대방 위젯이 바로 바뀌는 즉시성. 실시간 알림/반응 UX 참고.
- **수상/인지도:** 8,000만 다운로드, 일일 900만 활성 사용자. 100억 장 이상의 사진 공유.

### 1-4. BeReal
- **카테고리:** 일상 인증 사진 공유
- **플랫폼:** iOS, Android
- **참고할 디자인/기능:**
  - **듀얼 카메라 촬영 UI:** 전면+후면 동시 캡처 인터페이스. 독특한 촬영 UX 설계의 참고.
  - **시간 제한 알림:** 무작위 시간에 "지금 찍어!"라는 알림이 오는 참여 유도 메커니즘. Hearth의 리마인더/넛지 시스템 참고.
  - **무가공 철학:** 필터, 편집 없이 있는 그대로 공유. "진짜 순간"이라는 가치 설계 참고.
- **수상/인지도:** 2022 App Store "올해의 앱". 글로벌 월간 4,000만+ 사용자.

### 1-5. FamilyAlbum (みてね / Mitene)
- **카테고리:** 가족 사진 공유 앨범
- **플랫폼:** iOS, Android, Web
- **참고할 디자인/기능:**
  - **월별 자동 정렬 + 아이 나이 표시:** 사진이 시간순으로 자동 정리되며 아이의 나이가 함께 표시. Hearth의 타임라인 자동 구성 UX 참고.
  - **무료 월간 인화 서비스:** 매달 11장 무료 인화 배송. 인화 전환을 자연스럽게 만드는 비즈니스 모델 참고.
  - **초대 100명까지:** 대가족 단위 공유. 접근성이 높은 초대 흐름(기술에 약한 어르신도 쉽게 사용).
  - **3개월 컴필레이션 비디오:** 자동 생성되는 추억 영상. Hearth의 자동 콘텐츠 생성 참고.
- **수상/인지도:** 일본 1위 가족 앨범 앱. 글로벌 2,000만+ 다운로드.

## 2. 사진 편집 / 꾸미기 앱 (데코레이팅 UX 레퍼런스)

### 2-1. VSCO
- **카테고리:** 사진/비디오 편집 + 크리에이터 커뮤니티
- **플랫폼:** iOS, Android, Web
- **참고할 디자인/기능:**
  - **프리셋(필터) 시스템:** 200+ 큐레이션된 프리셋과 미세 조정 인터페이스. Hearth의 앨범 필터/무드 설정 UI 참고.
  - **VSCO Spaces:** 사진을 함께 모아보는 공유 갤러리. 공유 앨범과 커뮤니티의 중간 지점 UX 참고.
  - **좋아요/팔로워 숫자 없는 커뮤니티:** 숫자에 의한 압박 없이 순수하게 사진을 감상하는 피드 설계. Hearth의 소셜 피드 철학 참고.
  - **Canvas (무드보드 도구):** AI 기반 촬영 기획/무드보드 도구. 앨범 스타일 기획 기능의 참고.
- **수상/인지도:** 2018 Webby Award 수상. 200만+ 유료 구독자. 사진 커뮤니티의 사실상 표준.

### 2-2. Unfold
- **카테고리:** 스토리/콘텐츠 템플릿 에디터
- **플랫폼:** iOS, Android
- **참고할 디자인/기능:**
  - **컬렉션 기반 템플릿:** Film Frames, Ripped Paper, Journal, Clasico 등 테마별로 분류된 300+ 레이아웃. Hearth의 앨범 꾸미기 키트 구조 참고.
  - **폰트/스티커 마켓:** 독점 폰트와 스티커를 프리미엄으로 판매하는 모델. Hearth의 데코 아이템 마켓 비즈니스 모델 참고.
  - **드래그 앤 드롭 콘텐츠 배치:** 사진, 텍스트, 스티커를 직관적으로 배치하는 편집 UX.
- **수상/인지도:** Squarespace에 인수됨. Instagram 스토리 편집 앱 중 가장 높은 브랜드 인지도.

### 2-3. Picsart
- **카테고리:** AI 기반 사진/비디오 편집
- **플랫폼:** iOS, Android, Web
- **참고할 디자인/기능:**
  - **6,000만+ 스티커 라이브러리:** 사용자 생성(UGC) 스티커 생태계. Hearth의 스티커/데코 아이템 커뮤니티 기반 확장 참고.
  - **스티커 메이커:** 사용자가 직접 스티커를 만들어 공유하는 크리에이터 도구. Hearth의 사용자 제작 데코 아이템 기능 참고.
  - **레이어 기반 편집:** 전문적이면서도 접근 가능한 레이어 편집 UX. 앨범 페이지 꾸미기의 레이어 시스템 참고.
  - **AI 이미지 생성:** 텍스트 프롬프트로 배경/스티커 생성. Hearth의 AI 데코 기능 참고.
- **수상/인지도:** 월간 1.5억+ 활성 사용자. Getty Images 파트너십. 2024 Fast Company Most Innovative Companies 선정.

### 2-4. Canva
- **카테고리:** 올인원 디자인 도구 + 템플릿 마켓플레이스
- **플랫폼:** iOS, Android, Web
- **참고할 디자인/기능:**
  - **스크랩북 템플릿:** 전문 디자이너 제작 스크랩북 레이아웃. 비전문가도 퀄리티 높은 결과물을 만드는 템플릿 시스템.
  - **드래그 앤 드롭 + 가이드 스냅:** 요소 배치 시 자동 정렬 가이드. Hearth의 꾸미기 에디터 배치 도우미 참고.
  - **앱 마켓플레이스(Apps SDK):** 개발자/크리에이터가 Canva 안에서 앱을 만들어 판매. 플랫폼 확장성 설계 참고.
  - **매직 리사이즈:** 한 디자인을 다양한 사이즈로 자동 변환. 앨범 → 인화용 크기 변환 UX 참고.
- **수상/인지도:** Fast Company 2025 Innovation by Design Award 수상. 2.2억+ 월간 사용자.

### 2-5. Mojo
- **카테고리:** 애니메이션 소셜 미디어 콘텐츠 제작
- **플랫폼:** iOS, Android
- **참고할 디자인/기능:**
  - **애니메이션 템플릿:** 사진이 움직이는 스토리 템플릿. Hearth의 앨범 내 동적 요소(움직이는 스티커, 전환 효과) 참고.
  - **계정 없이 즉시 시작:** 가입 없이 바로 사용 가능한 온보딩. 진입 장벽 최소화 UX 참고.
  - **배경음악 내장:** 무료 사운드트랙 제공. 앨범에 음악 연결하는 기능 참고.
- **수상/인지도:** App Store 에디터 추천 다수. 소셜 미디어 크리에이터 도구 상위권.

## 3. 스크랩북 / 저널 / 다이어리 앱 (앨범 제작 UX 레퍼런스)

### 3-1. GoodNotes
- **카테고리:** 디지털 노트/저널/플래너
- **플랫폼:** iOS, iPadOS, macOS, Android, Windows
- **참고할 디자인/기능:**
  - **Apple Pencil 필기 + 팔레트 도구:** 자유로운 손글씨와 도형 그리기 인터랙션. Hearth의 앨범 위 직접 필기/낙서 기능 참고.
  - **노트북 커버 & 속지 커스터마이징:** 표지, 속지, 스티커를 조합하는 꾸미기 UX. 앨범 커버 디자인 흐름 참고.
  - **GoodNotes Marketplace:** 크리에이터가 노트 템플릿, 스티커, 플래너를 판매하는 마켓플레이스. Hearth의 데코 키트 마켓 구조 참고.
  - **PDF 위에 레이어 편집:** 기존 문서/이미지 위에 자유롭게 꾸미는 레이어 구조.
- **수상/인지도:** 2022 Apple Design Award 수상 (iPad). App Store "올해의 앱" 후보. 글로벌 노트 앱 1위.

### 3-2. Day One
- **카테고리:** 개인 일기/저널
- **플랫폼:** iOS, macOS, Android
- **참고할 디자인/기능:**
  - **멀티미디어 저널 엔트리:** 텍스트 + 사진 + 위치 + 날씨 + 오디오를 한 엔트리에 결합. Hearth의 앨범 페이지에 다양한 요소를 담는 구조 참고.
  - **"이 날의 기록" 리마인더:** 과거 일기를 자동 리마인드. Hearth의 추억 재방문 알림 참고.
  - **자동 메타데이터:** 위치, 날씨, 활동을 자동 기록. 사진에 맥락 정보를 자동 부여하는 UX 참고.
  - **프린트 연동:** 저널을 물리적 책으로 인쇄하는 기능. 디지털 → 피지컬 전환 UX 참고.
- **수상/인지도:** 2014 Apple Design Award 수상. App Store "에디터 추천" 상시.

### 3-3. Notion
- **카테고리:** 올인원 워크스페이스 / 노트
- **플랫폼:** iOS, Android, Web, macOS, Windows
- **참고할 디자인/기능:**
  - **블록 기반 편집 시스템:** 텍스트, 이미지, 임베드 등을 블록 단위로 자유롭게 조합. Hearth의 앨범 페이지 구성 요소를 블록처럼 배치하는 UX 참고.
  - **Template Gallery (30,000+):** 커뮤니티가 만든 방대한 템플릿 마켓. Hearth의 앨범/데코 템플릿 생태계 참고.
  - **커버 이미지 + 아이콘 커스텀:** 페이지마다 커버와 아이콘을 설정하는 간단한 개인화. 앨범별 커버 설정 UX 참고.
  - **다크모드/테마:** 라이트/다크 + 커스텀 테마. 앱 전체 테마 설정 참고.
- **수상/인지도:** 2021 Fast Company Innovation by Design 수상. 기업 가치 100억 달러+.

## 4. 한국 앱 UX 레퍼런스 (한국 시장 디자인 언어)

### 4-1. 토스 (Toss)
- **카테고리:** 금융 슈퍼앱
- **플랫폼:** iOS, Android
- **참고할 디자인/기능:**
  - **극단적 간결함의 UI:** 복잡한 금융을 한 화면에 핵심만 보여주는 미니멀 UX. Hearth의 복잡한 기능(꾸미기, 인화, 공유)을 간결하게 정리하는 설계 참고.
  - **바텀시트(Bottom Sheet) 인터랙션:** 모든 상세 기능을 바텀시트로 처리해 화면 이동을 최소화. 한국 사용자에게 익숙한 인터랙션 패턴.
  - **마이크로 애니메이션:** 송금 시 돈이 날아가는 등 의미 있는 마이크로 인터랙션. Hearth의 앨범 생성/꾸미기 시 만족감을 주는 애니메이션 참고.
  - **개인화 홈 화면:** 사용자별 맞춤 위젯/카드 배치. Hearth의 개인화 피드 설계 참고.
- **수상/인지도:** 2016 Google Play 올해의 앱(한국). 기업 가치 70억 달러. KPMG 세계 100대 핀테크.

### 4-2. 당근 (Karrot / 당근마켓)
- **카테고리:** 지역 기반 중고 마켓 + 커뮤니티
- **플랫폼:** iOS, Android
- **참고할 디자인/기능:**
  - **매너온도(Manner Temperature):** 36.5도에서 시작하는 신뢰도 시스템. Hearth의 사용자 간 신뢰/레벨 시스템 참고.
  - **동네생활(Neighborhood Life):** 거래 앱을 커뮤니티 앱으로 확장한 하이퍼로컬 피드. 사진 앱에서 소셜 커뮤니티로의 확장 전략 참고.
  - **극도의 심플한 네비게이션:** 기술에 약한 사용자(어르신)도 쉽게 사용하는 UX. 전 연령대 접근성 참고.
  - **위치 기반 필터링:** 반경 6km 기준의 지역 제한. Hearth의 위치 기반 포토스팟 기능 참고.
- **수상/인지도:** 한국 월간 활성 사용자 1,800만+. 한국 당 앱 사용시간 1위 커머스.

### 4-3. 오늘의집 (Ohouse)

  - **"For You" 자동 추억:** AI가 날짜/장소/인물을 인식해 자동으로 추억 컬렉션을 생성. Hearth의 AI 자동 앨범 생성 참고.
  - **Shared Albums (공유 앨범):** iCloud 기반 공유 앨범의 초대/권한/댓글 UX. iOS 사용자에게 이미 익숙한 패턴.
  - **스와이프 제스처:** 좌우 스와이프로 사진 넘기기, 위로 스와이프로 상세 정보 등. 사진 앱의 표준 제스처 규칙.
- **수상/인지도:** iOS 기본 앱. 사진 앱 UX의 사실상 표준.

### 8-2. Spotify Wrapped
- **카테고리:** 연간 리캡 경험 (Spotify 내 기능)
- **플랫폼:** iOS, Android
- **참고할 디자인/기능:**
  - **인터랙티브 스토리형 리캡:** 스와이프하며 넘기는 스토리 형태의 연간 요약. Hearth의 "올해의 앨범" / 연말 리캡 기능의 직접 참고.
  - **공유 최적화 디자인:** SNS 공유를 위해 최적화된 카드 디자인. 리캡 결과를 인스타그램/카카오톡으로 공유하는 바이럴 설계 참고.
  - **매년 바뀌는 비주얼 아이덴티티:** 2024년(팝 아트), 2025년(스트릿/그런지) 등 매년 새로운 디자인 테마. 시즌별/연도별 특별 테마 전략 참고.
  - **소셜 비교(Fan Leaderboard):** 친구들과 취향을 비교하는 소셜 요소. 친구 간 앨범 활동 비교/공유 참고.
- **수상/인지도:** 연말 글로벌 트렌드. 2024년에 10주년 기념. 매년 수억 건 SNS 공유.

### 8-3. Partiful
- **카테고리:** 이벤트/파티 초대장 플랫폼
- **플랫폼:** iOS, Android, Web
- **참고할 디자인/기능:**
  - **재미있는 커스텀 초대장 UI:** 패러디 앨범 커버, 테마 디자인 등으로 초대장을 꾸미는 UX. Hearth의 앨범 커버/초대 화면 꾸미기 참고.
  - **소셜 레이어(참석자 보기, 이모지 반응):** 콘텐츠에 소셜 인터랙션을 자연스럽게 얹는 방식. 앨범 참여자 표시/반응 UX 참고.
  - **GIF/비주얼 코멘트:** 텍스트 대신 GIF나 이모지로 반응하는 경량 소셜. Hearth의 앨범 댓글/반응 시스템 참고.
  - **텍스트 블래스트:** 참가자에게 일괄 업데이트를 보내는 기능. 공유 앨범 멤버에게 알림/공지를 보내는 UX 참고.
- **수상/인지도:** Google Play 2024 Best App. 20대 도시 거주 사용자 중심 성장.

### 8-4. Procreate / Procreate Dreams
- **카테고리:** 디지털 아트/애니메이션 제작
- **플랫폼:** iPadOS
- **참고할 디자인/기능:**
  - **멀티터치 + Apple Pencil 인터랙션:** 핀치, 탭, 스와이프를 활용한 유연한 캔버스 조작. Hearth의 앨범 꾸미기 캔버스 제스처 참고.
  - **브러시 라이브러리 구조:** 수백 개의 브러시를 카테고리별로 정리. 데코 도구(스티커, 브러시, 프레임)의 정리 UX 참고.
  - **커스텀 브러시 제작/공유:** 사용자가 브러시를 만들어 공유/판매. 크리에이터가 데코 도구를 제작하는 구조 참고.
  - **애니메이션 타임라인:** Procreate Dreams의 직관적 애니메이션 편집 UI. 앨범 내 동적 요소의 편집 인터페이스 참고.
- **수상/인지도:** 2024 Apple Design Award 수상 (Innovation). App Store iPad 앱 매출 상위.
